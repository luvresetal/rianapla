# rianapla

ここにはAndroid向けゲームアプリ「無限増殖リアナプラ」のソースデータ一式を置いてあります。
気が向いたときに更新してます。  

操作方法  
マウス左クリック：決定/リアナプラをつぶす  
  
あそびかた  
リアナプラ（オレンジ色のキャラ）はマウス左クリックでつぶすと3つに分裂するので  
ひたすらつぶして1分間のうちにどれだけ増やせるか競うゲームです。
